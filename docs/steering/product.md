# プロダクト概要

このファイル（`docs/steering/product.md`）を、プロダクト情報の正本
(canonical) とする。プロダクト内容を更新する場合は本ファイルを編集すること。

## 概要

SuteraVR は、仮想世界の持続性を実現することを目的とした分散型 Social-VR
である。従来は中央集権的な主体に集中していた運営コストと依存を、Federation
による分散と P2P による分散の二層構造で低減する。

### Federation による分散

Federation は、独立した複数ノードが共通プロトコルのもとで相互運用する仕組み
である。各ノードは独自ポリシーで運営でき、SuteraVR では主に以下を分散して担う。

- ワールドやアバターなど UGC の管理
- モデレーション

### P2P による分散

P2P はユーザー間で直接通信する仕組みである。SuteraVR では、ボイスチャットや
位置同期など高頻度データを主に P2P で処理し、クラウド依存とノード側の通信費
負担を低減する。必要に応じて STUN などのノード提供インフラを併用する。

## コア能力

- Bloom による WebRTC シグナリング
  (ルーム作成/参加/離脱、Offer/Answer/ICE 中継)
- Syncer による P2P 同期
  (Pose/Chat/Voice/Control など高頻度データ)
- Federation ノード間の相互運用による運営分散
- VRChat との部分的互換性を視野に入れたプロトコル設計

## 主なユースケース

- 複数コミュニティによる分散運営の仮想世界
- 低レイテンシな音声/位置同期を必要とする Social-VR
- 中央集権に依存しない長期運用の仮想空間

## プロダクト制約

- VRChat との部分的互換性を重視する
- ただし、SuteraVR の理念と目的を損なわない範囲で実現する

## Mission / Vision / Value

- Mission: 想われる限り、朽ちぬ仮想世界を
- Vision: 明日も、あなたへ「月が綺麗ですね」と伝えられますように
- Value: 創造者の手を離れ、自ら芽吹く世界に

## 価値提案

- 運営主体の単一障害点を排除し、世界の持続性を担保
- P2P により通信コストを抑え、スケールしやすい
- 「創造者の手を離れても続く世界」を実現する基盤
