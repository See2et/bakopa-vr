# アーキテクチャ文書の参照先

このファイルは移行用の入口です。アーキテクチャ情報の正本は、用途別に
`docs/steering/` 配下へ統合しています。

- 技術方針・プロトコル・運用制約: `docs/steering/tech.md`
- 構造・責務分割・ディレクトリ方針: `docs/steering/structure.md`
- プロダクト背景・価値提案: `docs/steering/product.md`

時点依存の進捗・実装ステータスは steering ではなく、必要に応じて
`docs/specs/` の仕様・検証ドキュメントで管理すること。
