# BakopaVR
This would be a minimal (value) product of SuteraVR.
Bakopa is another name of sutera flower.

## Development

Rust tooling is provided via Nix Home Manager, so `cargo` and `rustc` are already
available without installing `rustup`. When running commands such as `cargo test`
or `cargo fmt`, make sure your shell session has the Nix environment loaded (for
example by sourcing `~/.nix-profile/etc/profile.d/hm-session-vars.sh` or entering
the project through `nix develop`).
